Charity Finder

Project started: August 6, 2025 (8/6/2025)

Project type: Downingtown West High School graduation project

About this project

This website — Charity Finder — was built as a graduation project by students at Downingtown West High School. Its goal is to help users easily find the right charity to support by providing a searchable, organized list of charities along with descriptions, keywords, and links to their websites.

This project was created by

  • Evan O
  • Nathan P
  • Frederick B

Purpose

The purpose of this site is to help people find the charity that is right for them. By organizing charities with helpful descriptions, keywords, and direct links, this project aims to make it easier for anyone to connect with causes they care about.
